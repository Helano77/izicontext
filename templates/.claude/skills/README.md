# Skills

Step-by-step guides for recurring patterns in this service.

## Available Skills

| Skill | When to use |
|-------|------------|
| `go-patterns/` | Go-specific patterns: constructors, error handling, tests, imports |
| `service-communication/` | Calling other services: HTTP clients, events, timeouts |
| `bug-reproduction/` | Reproducing and isolating bugs |
| `batch-operations/` | Large refactors across multiple files |
| `git-platform/` | Detecting git platform (GitHub/GitLab) for CLI commands |

## Adding a Skill

Run `/add-skill` in Claude Code for interactive creation.

Skills are user-managed — `izicontext update` will never overwrite them.
